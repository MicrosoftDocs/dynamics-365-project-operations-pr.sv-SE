# <a name="resource-fulfillment-for-planned-resources"></a><span data-ttu-id="c244b-103">Resursuppfyllelse för planerade resurser</span><span class="sxs-lookup"><span data-stu-id="c244b-103">Resource fulfillment for planned resources</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="c244b-104">En projektledare kan planera nödvändiga resursroller för ett projekt.</span><span class="sxs-lookup"><span data-stu-id="c244b-104">A project manager can plan required resource roles for a project.</span></span> <span data-ttu-id="c244b-105">Resurshanteraren visar de här planerade resurserna som förfrågningar på sidan **Resursuppfyllelse** och kan tilldela verkliga resurser.</span><span class="sxs-lookup"><span data-stu-id="c244b-105">The resource manager will see these planned resources as requests on the **Resource fulfillment** page and can assign actual resources.</span></span>

1. <span data-ttu-id="c244b-106">På sidan **Alla projekt** välj **XYZ uppgradera fas 2** projekt.</span><span class="sxs-lookup"><span data-stu-id="c244b-106">On the **All projects** page, select the **XYZ Upgrade Phase 2** project.</span></span>
2. <span data-ttu-id="c244b-107">Välj **Projekt** och välj sedan **Redigera**.</span><span class="sxs-lookup"><span data-stu-id="c244b-107">Select **Project**, and then select **Edit**.</span></span>
3. <span data-ttu-id="c244b-108">På sidan **Projektgrupp och schemaläggning** välj **Lägg till**.</span><span class="sxs-lookup"><span data-stu-id="c244b-108">On the **Project team and scheduling** tab, select **Add**.</span></span>
4. <span data-ttu-id="c244b-109">I dialogrutan **Lägg till roller** välj rollen **Programvaruutvecklare**.</span><span class="sxs-lookup"><span data-stu-id="c244b-109">In the **Add roles** dialog box, select the **Software developer** role.</span></span>
5. <span data-ttu-id="c244b-110">Klicka på **Skapa** och stäng sedan projektsidan.</span><span class="sxs-lookup"><span data-stu-id="c244b-110">Select **Create**, and then close the project page.</span></span>
6. <span data-ttu-id="c244b-111">På sidan **Resursuppfyllelse** väljer du **Programvaruutvecklare 1** för projektet **XYZ uppgradering projektfas 2**.</span><span class="sxs-lookup"><span data-stu-id="c244b-111">On the **Resource fulfillment** page, select **Software developer 1** for the **XYZ Upgrade project Phase 2** project.</span></span>
7. <span data-ttu-id="c244b-112">Välj en arbetare och sedan **Tilldela**.</span><span class="sxs-lookup"><span data-stu-id="c244b-112">Select a worker, and then select **Assign**.</span></span>
8. <span data-ttu-id="c244b-113">Kontrollera att raden för **programvaruutvecklare 1** har tagits bort för projektet **XYZ uppgradering projektfas 2**.</span><span class="sxs-lookup"><span data-stu-id="c244b-113">Verify that the line for **Software developer 1** has been removed for the **XYZ Upgrade project Phase 2** project.</span></span>
9. <span data-ttu-id="c244b-114">På fliken **Projektteam och schemaläggning** för projekt **XYZ uppgradera fas 2**, kontrollera att arbetaren som du valde i föregående steg har lagts till som **Programvaruutvecklare**.</span><span class="sxs-lookup"><span data-stu-id="c244b-114">On the **Project team and scheduling** tab, for the **XYZ Upgrade Phase 2** project, verify that the worker that you selected in the previous step has been added as **Software developer**.</span></span>
